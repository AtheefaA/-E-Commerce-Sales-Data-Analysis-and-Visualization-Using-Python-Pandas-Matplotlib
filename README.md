# 📊 E-Commerce Sales Data Analysis and Visualization

## 📖 Project Overview

This project demonstrates a complete data analysis workflow using **Python**, **Pandas**, and **Matplotlib**. The application loads sales data from a CSV file, cleans the dataset, performs analysis, and creates visualizations to provide meaningful business insights.

The project showcases data loading, preprocessing, analysis, visualization, and reporting in a simple and beginner-friendly manner.

---

## 🎯 Objectives

* Load sales data from a CSV file
* Clean and preprocess the dataset
* Handle missing values and duplicate records
* Calculate key sales metrics
* Create professional data visualizations
* Generate meaningful business insights

---

## 🛠️ Technologies Used

* Python 3
* Pandas
* Matplotlib

## 🚀 Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/E-Commerce-Sales-Data-Analysis-and-Visualization.git
```

### 2. Navigate to the Project Folder

```bash
cd E-Commerce-Sales-Data-Analysis-and-Visualization
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Project

```bash
python main.py
```
