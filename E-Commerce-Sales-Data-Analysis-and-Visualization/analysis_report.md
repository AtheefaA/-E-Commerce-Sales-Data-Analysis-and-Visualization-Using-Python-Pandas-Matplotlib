# 📊 E-Commerce Sales Data Analysis Report

## 1. Project Overview

The objective of this project is to perform complete sales data analysis using Python, Pandas, and Matplotlib. The project demonstrates data loading, cleaning, analysis, visualization, and reporting using a real-world style dataset.

---

## 2. Objectives

* Load data from CSV
* Clean the dataset
* Handle missing values
* Remove duplicate records
* Calculate important sales metrics
* Generate visualizations
* Write meaningful insights

---

## 3. Tools & Technologies

* Python
* Pandas
* Matplotlib
* VS Code
* GitHub

---

## 4. Setup Instructions

1. Install Python

2. Install required libraries:

```bash
pip install -r requirements.txt
```

3. Run:

```bash
python main.py
```

## 5. Code Structure
